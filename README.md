# WebHook-Azure-Devops-SSMS-SLACK-DOCKER-
Azure DevOps Work Item Event’lerini dinleyip tüm değişiklikleri (create/update/delete) otomatik olarak yakalayan, SQL Server üzerinde güncel durumu tutan, tarihçe kayıtlarını yöneten, Slack ile entegre çalışan ve Docker ortamında 7/24 stabil şekilde çalışmak üzere tasarlanmış gerçek zamanlı bir iş takip ve entegrasyon botu.
